## Contributing

To add a new language translation:

1. Copy `README.md` (English) as a template
2. Translate UI text, descriptions, and documentation sections
3. Keep all technical content unchanged (code, commands, paths, URLs)
4. Maintain exact markdown formatting
5. Save as `docs/locales/README.<language-code>.md`
6. Submit a pull request
